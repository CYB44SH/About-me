# About-Me

This document includes summary of the certifications and courses I have completed.

# Network Fundamentals Certificate
In this course, I learned about different network tools and how they are used, as well as the various network protocols and how each protocol functions within its corresponding layer.

# NASA Space Hackathon Certificate

The NASA certificate related to the Space Hackathon was an exceptional experience that significantly enhanced my practical skills. I competed alongside experts and researchers in space science and technology, and our team was selected from among 1,000 participants.

During the hackathon, we developed a prototype that supports children by teaching them self‑reliance through an interactive farming concept where they care for plants. We connected the idea to space by importing real data and integrating it into the game. The weather inside the game updates dynamically based on the space‑related data retrieved through an API.

# malware forensics
This workshop was one of the best learning experiences I’ve attended. I learned how to analyze malware, understand its behavior and signatures, and use several analysis tools and commands—including the "strings" command—to extract readable text, gather useful information, and identify key indicators within the malware sample. I also used VirusTotal, a multi‑engine malware scanning platform, to check files and URLs and review shared analysis reports from the security community.

The workshop also explained the limitations of traditional antivirus solutions, as they rely heavily on predefined signatures, which makes frequent updates necessary to keep up with evolving threats. Additionally, it introduced platforms where malware analysts share their findings, techniques, and analysis reports with the community.
